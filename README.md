# Mini Healthcare Support Helper

## Tech Stack
- React + Vite (Frontend)
- Node.js + Express (Backend)
- External AI API
- Vercel (Frontend Hosting)
- Render (Backend Hosting)

## Features
- Patient support intake form
- Automatic patient case summary
- AI-powered FAQ chatbot
- Secure backend AI API proxy
- Clean NGO-style dashboard UI
- Responsive two-column layout

## AI / Automation Idea
An AI chatbot answers common healthcare support questions and reduces NGO volunteer workload. The system also auto-generates structured case summaries from form inputs.

## NGO Use Case
Helps NGOs quickly register patient support requests and provide immediate AI-based guidance before human follow-up.
